# Digital Learning Management System
### PROG 102 — Principles of Software Engineering
**Limkokwing University of Creative Technology — Sierra Leone**

## Project Overview
A Digital Public Good mobile application designed to improve 
access to quality education in Sierra Leone.
Aligned with UN SDG 4: Quality Education.

## Features
- Student & Teacher login
- Course management
- Learning material upload & download
- Online quizzes & assessments
- Result viewing

## SDG Alignment
SDG 4 — Quality Education
